# Proyecto Gaming

## Descripción
Este proyecto presenta un sitio web sobre el mundo de los videojuegos, destacando desarrolladoras como Nintendo y Rockstar Games, junto con algunos de sus títulos más famosos.

## Acciones correctivas
- Eliminación de estilos en línea (`style=""`).
- Centralización de todos los estilos en un archivo CSS externo.
- Corrección de estructura HTML para mayor claridad y semántica.

## Acciones evolutivas
- Mejora de tipografía y legibilidad.
- Estilización de secciones (`article` y `aside`) con fondos diferenciados y sombras.
- Imágenes responsivas, centradas y con efecto hover.
- Menú de navegación centrado, con transiciones suaves y colores consistentes.
